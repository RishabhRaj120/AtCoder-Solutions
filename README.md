## About This Repository

This repository is my personal archive of **AtCoder solutions**, written mainly in **C++**.

I use it to track my competitive programming journey, revise solved problems, and improve problem-solving speed, accuracy, and consistency.

---

## Folder Structure

Solutions are organized by **contest series and contest number**:

```text
AtCoder-Solutions/
+-- AtCoder Beginner Contest (ABC)/
|   +-- ABC_297/
|   |   +-- A_Double_Click.cpp
|   +-- ...
+-- README.md
```
